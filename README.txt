===================
cookiecutter-tryton
===================

Cookiecutter_ template for a NaN-tic's Tryton module

Usage
-----

To install::

    mkdir ~/.cookiecutters
    cd ~/.cookiecutters
    git clone git@github.com:nan-tic/cookiecutter-tryton tryton
    pip install cookiecutter

To use::

    cd /path/to/your/tryton/modules
    cookiecutter tryton
